Is your social media post toxic or not?

If it is, then what kind of toxic comment is it?


Social media sites have become an avenue for some users to spread hate speech and toxic comments. Others are harassed and become victims of cyberbullying.

The goal of this project is to use deep learning methods to identify toxicity in online comments, which can be used to filter potentially harmful messages, promote a healthier and safer social media space for everyone especially for the kids, and to help users craft more civil arguments when engaging in an online discussion. This project used deep learning models to classify a certain comments as toxic, severely toxic, identity hate, insult, obscene or threat.

Use this web application to check and identify the toxic comment characteristics of your social media post!
